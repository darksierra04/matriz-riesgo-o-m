# Matriz de Riesgo O&M

Dashboard web (HTML/CSS/JS) para registrar y visualizar riesgos de O&M por tipo de proyecto.

## Demo
- GitHub Pages: (https://darksierra04.github.io/matriz-riesgo-o-m/)

## Funcionalidades
- Filtro por tipo de proyecto (TODOS, AGPE, AGGE, GD, Utility Scale)
- KPIs: total, bajo, medio, alto, crítico, efectividad de mitigación
- Distribución por categoría

## Cómo editar
- `index.html`: estructura + estilos
- `logo-greenyellow1.png`: logo

## Notas
- Sitio estático (sin backend). Los datos se manejan en el navegador.
